# Local Development

## What this is

Run the app on your own machine while you build it.

## How to start

1. Open the project folder.
2. Run a local web server from the repo root.
3. Open `http://127.0.0.1:8000/` in the browser.

## What to check

- The workspace tabs show up.
- You can switch between workspaces.
- The widgets load without errors.
- Local changes stay in the browser until you export them.

## Private data rule

- Keep private config in `config.local.js`.
- Keep personal backups out of Git.
